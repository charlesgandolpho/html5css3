One Page Site

Ideal para Negocios de pequeno e medio porte o "one page site" é um site onde voce inclui todas as informacoes que o visitante precisa de forma objetiva criativa em uma unica pagina.

Wireframe

Wireframe é uma representacao visual da estrutura básica de uma pagina. Ele deve ser feito de maneira mais simples possivel, mostranto somente o essencial, como uma especie de rascunho, sem cores ou imagem. Clique no botao ou escaneie o QR Code para obter o wireframe desta pagina.
